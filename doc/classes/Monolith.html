Monolith.InitOptions
InitOptions

Fields:

  • resolution table Game resolution. (default {128 128})
  • windowScale number Window scale (default 1)
  • targetFps number Target frame rate (default 60)
  • useCanvas boolean If true, use canvas, else use screenshot (default true)
  • ledSize table LED matrix board unit size (default {64 64})
  • ledChains number LED matrix chains size (default 4)
  • ledParallels number LED matrix parallels size (default 1)
  • ledPixelMapper string LED matrix mapper name (default 'Z-mapper')
  • ledBrightness number LED matrix brightness, range: 0.0 ~ 1.0 (default 1.0)
  • ledColorBits number LED matrix size of color bit, range: 1 ~ 8 (default 8)
  • ledResetChip boolean Reset LED matrix chip is enabled (default true)
  • inputEnabled boolean Input field is enabled (default true)
